# any-console

轻量的用于移动设备真机调试的虚拟控制台

Console for mobile devices.

## 安装 Install
`npm i any-console -S`

## 使用 Usage

浏览器中使用 Browser
```
// html
<script src="any-console.js"></script>

// js
new AnyConsole();
```

commonjs
```
const ac = require('any-console');

new ac();
```

es module
```
import ac from 'any-console';

new ac();
```
